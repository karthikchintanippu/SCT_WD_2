# Key Features

## Accurate Stopwatch Engine

The stopwatch uses JavaScript timing functions to ensure reliable and precise tracking.

Features include:

- Start timer
- Pause timer
- Resume timer
- Reset timer
- Real-time updates

---

## Millisecond Precision

The timer tracks:

- Minutes
- Seconds
- Centiseconds

This provides greater accuracy than a simple second-based stopwatch.

---

## Lap Tracking System

Users can record multiple laps during a session.

Each lap stores:

- Total elapsed time
- Split time since the previous lap

The latest lap appears at the top of the list for quick viewing.

---

## Fastest & Slowest Lap Detection

ChronoGraph automatically identifies:

- Fastest lap (highlighted in green)
- Slowest lap (highlighted in red)

The indicators update dynamically as new laps are recorded.

---

# Core Functionality

| Feature | Description |
|----------|-------------|
| Start | Begins stopwatch timing |
| Pause | Stops timing without losing progress |
| Resume | Continues from paused state |
| Reset | Clears timer and laps |
| Lap | Records current timing data |
| Split Time | Measures interval between laps |
| Fastest Lap | Highlights shortest lap |
| Slowest Lap | Highlights longest lap |
